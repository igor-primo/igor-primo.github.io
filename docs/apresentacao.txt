. Aumento exponencial de geração de dados -> Big Data
. Impacto de Big Data em todos os setores.
  - Produção diária de 2,5 quintilhões de bytes.
  - Existência estimada de mais de 40 sextilhões de dados
    (zettabytes)
  - Estruturados, semi-estruturado, não-estruturados,
    além de vídeos, imagens e áudios.
. Bancos de dados relacionais e data warehouses
  não tratam bem o alto volume de fluxo de dados
. Exemplos de campos impactados, segundo literatura
  - saúde
  - predição de desastres naturais
  - Experts argumentam que para melhor explorar
    a utilidade de BD, é necessário utilizar tecnologias

    - DevOps + adaptação para análise de dados
      - Redução de tempo e esforço na aquisição de dados
        - DataOps
          - Integra pessoas, processos e tecnologias para
            possibilitar o manuseio ágil e automatizado
